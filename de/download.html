Lade den Chord Scale Generator kostenlos herunter

Es stehen Versionen für die Betriebssysteme Windows, Mac OS X (Snow Leopard oder Lion) und Linux zur Verfügung. Zusätzlich kannst du bei den Versionen für Windows und Linux zwischen einer 32- und 64-bit Variante wählen.

Windows 10 (32-bit) | Windows 10 (64-bit)

Windows 7 / Vista / XP (32-bit) | Windows 7 / Vista / XP (64-bit)

Mac OS X (64-bit)

Linux (32-bit) | Linux (64-bit)

Hinweis für Windows 10: Zip-Datei entpacken (z.B. in C:\Download(s) oder in C:\Eigene Dokumente\CSG ... nur NICHT in C:\Programme, dort braucht man besondere Schreibrechte). Dann .exe starten. Um eine Verknüpfung mit dem Desktop einzurichten: Rechte Maustaste auf .exe > Senden an > Desktop (Verknüpfung erstellen).

Um den Chord Scale Generator als Vollversion zu nutzen, benötigst du einen Aktivierungsschlüssel. Seit Anfang 2021 ist der Kauf eines solchen Schlüsels nicht mehr notwendig, die Nutzung der Software ist komplett frei. Im Aktivierungsdialog kannst du daher einen beliebigen Code angeben.
