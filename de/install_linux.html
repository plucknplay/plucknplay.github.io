Installationsanleitung – Linux Windows Mac OS X

1. Lade (sofern notwendig) die Java Runtime Environment herunter und installiere diese

Damit du den Chord Scale Generator ausführen kannst, benötigst du die Java Runtime Environment (JRE) in der Version 6. Falls auf deinem Rechner diese noch nicht installiert ist, lade dir die neueste Version unter dem folgenden Link kostenlos herunter:

Oracle: Java SE Downloads

Nach Abschluss des Downloads installiere die Java Runtime Environment auf deinem Rechner.

2. Lade den Chord Scale Generator herunter

Lade dir ebenfalls den Chord Scale Generator herunter.

CSG-linux-gtk.tar.gz bzw. CSG-linux-gtk_64.tar.gz

Merke dir dabei, wo die Datei gespeichert wurde.

3. Entpacke die heruntergeladene Archiv-Datei

Nach Abschluss des Downloads entpacke die heruntergeladene Archiv-Datei in den gewünschten Zielordner. Öffne hierfür eine Konsole und nutze folgenden Befehl, um die Datei zu entpacken:

tar -C /ziel_ordner -xvf CSG-linux-gtk.tar.gz

4. Starte den Chord Scale Generator

Nach Abschluss der obigen Installationsschritte kannst du den Chord Scale Generator nun starten. Führe hierfür die im Installationsordner befindliche ausführbare Datei aus.

Beim Start wird dir zunächst der Dialog zur Aktivierung der Vollversion angezeigt. Solltest du bereits einen Aktivierungsschlüssel erworben haben, kannst du die Vollversion gleich aktivieren. Du kannst natürlich die Aktivierung auch überspringen, um zunächst die Test-Version des Chord Scale Generators auszuprobieren.
Hinweis: Zum Aktivieren der Vollversion benötigst du kurzzeitig einen Internetzugang.

Beim ersten Start werden dir zudem die Willkommens-Seiten gezeigt. Wir empfehlen dir, diese durchzulesen. Besonders die Rundtour gibt dir eine erste ausführliche Beschreibung des Funktionsumfangs dieses Programms.
