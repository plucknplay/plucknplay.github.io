Download Your Free Version

The Chord Scale Generator is available for the operating systems Windows, Mac OS X (Snow Leopard or Lion) and Linux. Besides, you can choose between 32- and 64-bit versions for Windows and Linux.

Windows 10 (32-bit) | Windows 10 (64-bit)

Windows 7 / Vista / XP (32-bit) | Windows 7 / Vista / XP (64-bit)

Mac OS X (64-bit)

Linux (32-bit) | Linux (64-bit)

Instructions for Windows 10: Decompress the downloaded zip file in a folder of your choice (e.g. C:\Download(s) or C:\apps\CSG ... DO NOT use C:\Program Files\, you would need special file writing rights there). Then start the .exe file. For an icon on your Windows desktop: Right mouse click on .exe > Send to > Desktop (create shortcut).

In order to use the full version of the Chord Scale Generator you need an activation key. Since early 2021 we no longer require to purchase such a key, the software can be used for free. Thus, in the activation dialog any code will work.
