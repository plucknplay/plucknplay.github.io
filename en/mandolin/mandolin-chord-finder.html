Mandolin Chord Finder

The chord finder feature of the award-winning Chord Scale Generator helps you to find chords that fit a certain fingering pattern or musical notation. This software tool supports the most common mandolin tunings as well as any other string instrument tuning.

Chord Name Finder

The reverse chord finder helps you to identify the name(s) of a given chord. Chords without a root note, 3rd or 5th can optionally be included in this search, too.

Translate Between Notations

Don't know a certain notation? This software tool lets you translate between musical notation, tablatures and chord charts and thus helps you to get familiar with unknown notations.

Find Alternative Fingerings

Based on the musical notation of a chord, easier alternative fingerings can be found.

80+ Types of Chords

The reverse chord finder and chord name finder features are based on an extendable list of 80 types of chords and thus cover thousands of chords.
