I have installed the Chord Scale Generator but I can't start it?

In order to run the Chord Scale Generator you need to have a Java Runtime Environment (JRE) installed on your computer. The minimum version required is 6 (internal version number 1.6.0). Here you can download the current version of the JRE for free:

Oracle: Java SE Downloads

What does the abbreviation 1.x stand for?

Currently, you can download the Chord Scale Generator in version 1.3. However, the activation key which you can purchase on this website is also valid for all minor releases that might be developed in the future (such as 1.4, 1.5, ...). New minor releases will be made accessible via (automatical) program updates.
However, if any major releases (such as 2.0, 3.0, ...) should be developed, a small upgrade fee might be charged.

How can I determine whether my operating system is 64-bit compatible?

Windows: In order to determine whether a 32- or 64-bit version of your Windows system is running, simply follow the instructions described in the Microsoft Support Pages.

Linux: To determine the bit count under Linux use the following command:

uname -a

In case of a 64-bit platform the result of the command will contain 'x86_64' or 'ia64', respectively.
